# NFT Trading Bot (Personal Use)

This project is a personal NFT trading bot for Ethereum and L2 networks.
The goal is to automate buying and listing NFTs according to my own strategy.

Features planned:
- Automated offers based on floor price
- Automatic listing after purchase
- Risk management and max budget control
- Reacting to price competition (undercut)

Starting budget for testing: 0.1 ETH
This bot is for personal use only.
